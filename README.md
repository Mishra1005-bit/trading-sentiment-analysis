# Trading Behavior Analysis based on Market Sentiment

## Overview

This project analyzes trading performance based on market sentiment (Fear vs Greed).

## Steps Performed

* Cleaned datasets (no missing/duplicates)
* Converted timestamps to datetime
* Aggregated data on daily level
* Merged datasets using date
* Created key metrics:

  * Win rate
  * Daily PnL
  * Average trade size

## Key Insights

* Fear days show better profitability
* Greed days show lower win rate due to aggressive trading
* Extreme Greed has higher win rate but controlled profits

## Strategy Suggestions

* Trade selectively during Fear periods
* Avoid overtrading during Greed
* Maintain balanced risk during Extreme Greed
